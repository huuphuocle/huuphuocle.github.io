Google Summer of Code 2022

Sampling correlation matrices

During June-September 2022, I collaborated with GeomScale organization to contribute to the open-source statistical software Volesti.
I implemented new classes and geometric algorithms for sampling correlation matrices from a truncated distribution.

This implementation provides a much faster functionality for this sampling problem in Volesti.

Maple libraries

ParamHermite

A Maple library for solving parametric polynomial systems and real quantifier elimination.
This implementation is based on our paper "Solving parametric systems of polynomial equations over the reals through Hermite matrices".

GitLab

It is used to classify the followng parametric polynomial system from Kuramoto model of 4 oscillators




IsolatedPoints

A Maple library for computing real isolated solutions of a multivariate polynomial equation.
